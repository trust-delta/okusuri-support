# 推奨コマンド

## 開発サーバー起動
```bash
npm run dev
```

## ビルド
```bash
npm run build
```

## 本番サーバー起動
```bash
npm start
```

## Linting
```bash
npm run lint
```

## フォーマット
```bash
npm run format
```

## Convex開発モード
```bash
npx convex dev
```

## Convexデプロイ
```bash
npx convex deploy
```

## 型チェック
```bash
npx tsc --noEmit
```

## Git操作
```bash
git status
git add <files>
git commit -m "message"
git push
```
