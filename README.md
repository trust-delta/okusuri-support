## プロジェクト概要

### 解決する課題

患者の**服薬アドヒアランス低下**（服薬忘れ・中断による症状悪化）の解決

### ターゲットユーザー

- **主利用者**: 精神疾患を持つ患者（独力での服薬管理が困難な方）
- **支援者**: 患者の家族・介護者（1患者あたり0-3名想定）
