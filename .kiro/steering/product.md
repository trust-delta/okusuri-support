# Product Steering Document

## Product Overview
**おくすりサポート** は精神疾患患者の服薬アドヒアランス向上を目的とした Progressive Web Application (PWA) 型の服薬管理アプリケーションです。

患者本人と支援者（家族・介護者）が協働して服薬管理を行えるプラットフォームを提供し、服薬忘れや中断による症状悪化を防ぎます。

## Core Features
- **服薬リマインダー**: 定期的な服薬時間の通知とアラート
- **服薬記録管理**: 服薬履歴の記録と可視化
- **支援者連携**: 家族・介護者による服薬状況のモニタリング（1患者あたり1-3名）
- **認証機能**: Clerkを使用したセキュアなユーザー認証
- **リアルタイムデータ同期**: Convexによるリアルタイム同期機能
- **PWA対応**: オフライン機能とネイティブアプリライクな体験

## Target Use Case
### 主利用者（患者）
- 精神疾患を持ち、独力での服薬管理が困難な方
- 定期的な服薬が必要だが、忘れやすい方
- デジタルツールを使った管理に抵抗がない方

### 支援者（家族・介護者）
- 患者の服薬状況を遠隔で確認したい家族
- 複数の患者を管理する介護施設のスタッフ
- 服薬忘れの際に迅速にサポートしたい支援者

### 利用シナリオ例
1. **日常の服薬管理**: 患者が毎日決まった時間に通知を受け、服薬を記録
2. **服薬忘れ対応**: 服薬未記録時に支援者へアラート送信
3. **定期診察時の情報共有**: 服薬履歴を医療従事者と共有
4. **緊急時対応**: 服薬状況の異常を早期発見

## Key Value Proposition
### 患者にとっての価値
- **服薬継続率の向上**: リマインダーと記録機能による習慣化支援
- **自立性の維持**: 自己管理能力の向上と主体性の確保
- **安心感の提供**: 支援者との繋がりによる心理的安定

### 支援者にとっての価値
- **負担軽減**: 遠隔モニタリングによる物理的・心理的負担の削減
- **早期介入**: 服薬状況の異常を即座に検知し対応可能
- **コミュニケーション向上**: 服薬データを基にした建設的な対話

### 社会的価値
- **医療費削減**: 服薬アドヒアランス向上による再入院率低下
- **QOL向上**: 患者と支援者双方の生活の質向上
- **データ活用**: 服薬パターンの分析による医療サービス改善

## Product Goals
- **Phase 1**: MVP開発 - 基本的な服薬記録と通知機能
- **Phase 2**: 支援者機能 - マルチユーザー対応と権限管理
- **Phase 3**: 高度な分析 - AI活用による服薬パターン分析とレコメンデーション
- **Phase 4**: 医療連携 - 医療機関との情報連携基盤構築

## Success Metrics
- **服薬継続率**: 90%以上の服薬実施率
- **ユーザー満足度**: NPS 40以上
- **支援者エンゲージメント**: 週3回以上のアクセス
- **システム信頼性**: 99.9%以上の稼働率
